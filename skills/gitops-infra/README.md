# GitOps Infra Executor Skill

**Description**: Executes infrastructure tasks including builds, deployments, configs.

## Inputs
- **Task ID**: e.g., "TASK-001"
- **Task File Path**: e.g., "tasks/active/TASK-001.md"

## Outputs
- **Execution Report**: Status of the execution.
- **Modified System**: Built images, updated manifests.

## Usage
Provide the Task ID and file path to the agent.
